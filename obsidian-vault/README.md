# Obsidian Second Brain

A minimal knowledge management vault for software project management.

## Structure (PARA Method)

| Folder | Purpose |
|--------|---------|
| `00 - Inbox` | Quick capture, unsorted notes |
| `10 - Daily` | Daily notes (YYYY-MM-DD.md) |
| `20 - Projects` | Active project folders |
| `30 - Areas` | Ongoing responsibilities |
| `40 - Resources` | Reference material |
| `50 - Archive` | Completed projects |
| `Templates` | Note templates |

## Quick Start

1. Open this folder in Obsidian
2. Set `Templates/` as your template folder in Settings → Core Plugins → Templates
3. Use **Cmd+P** → "Insert template" for new notes
4. Daily notes go in `10 - Daily/` with format `YYYY-MM-DD.md`

## Conventions

- Use `[[wikilinks]]` for cross-references
- Tags: `#decision`, `#blocker`, `#meeting`, `#idea`
- Archive completed projects to `50 - Archive/`
